

## 直線上界址標示


 ### [臺中市新聞簡介](https://www.taichung.gov.tw/2831673/post)

### 說明：
使用於[土地複丈規費服務網](https://easymap.land.moi.gov.tw/BSWeb/)瀏覽器擴充功能
於試算規費查詢時，判斷若「界址點坐標」與「前後界址點連線」之垂距小於自行設定之容許值，則視為在同一條直線上，並將該界址點號顏色改為紅色，以做為民眾選擇界址點時之參考。
